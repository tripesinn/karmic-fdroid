# Karmic Gochara — F-Droid Repository

F-Droid self-hosted repository for **Karmic Gochara Lite**.

## Install via F-Droid

1. Open F-Droid
2. Go to **Settings → Repositories → +**
3. Add this URL:

```
https://tripesinn.github.io/karmic-fdroid/repo
```

4. Search for **Karmic Gochara Lite** and install.

## About

Karmic Gochara Lite is a Vedic astrology transit calculator using the DK sidereal system.  
Calculations are performed locally using Swiss Ephemeris — no data sent to any server.
